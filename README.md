# Modular Life Sim — Website

Official landing page for the Modular Life Sim project.

## Overview

This repository contains the public-facing website for **Modular Life Sim**, an open-ended evolutionary simulation exploring multi-scale emergence, persistent identity, and collective intelligence.

- `lifesimdemo6x.mp4` — Demo video
