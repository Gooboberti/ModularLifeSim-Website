How the Brain Works in the Main Simulation

Architecture used in the full Modular Life Sim

Core Features

  • Structural Evolution — Networks can grow new neurons and prune weak ones across generations.
  • Parallel Pathways — Two computation paths (Module_A and Module_B) that can be dynamically weighted.
  • Recurrent Memory — GRU-style hidden state that integrates information over time.
  • Module Influence — Physical modules directly affect how the neural network develops.
  • Heritable Emotional Sensitivities — Traits like fear, curiosity, and aggression are inherited and mutated.

In This Mini-Game (v0.2)

This simplified version visualizes a reactive neural network with:

  • 3 inputs (Food angle, distance, energy)
  • 6 hidden neurons with recurrent memory feedback
  • Parallel pathway influence (visualized as two groups of connections)
  • 2 outputs (Turn + Forward speed)

The full simulation uses a much more complex evolving architecture with structural changes over generations.
