Real Data Charts (Completed)

v1.4
Clickable Trends + Expanded Analytics Modal
• Clicking the small Trends chart opens a large centered Simulation Analytics modal
• Modal shows live stats + large chart area with line toggles
• Foundation for future mini-graph line selection
v1.2 – v1.3
UI Polish & Menu Placement
• Aerosol Sprays button moved from sidebar to bottom action bar
• Header text cleaned and version clearly displayed in upper left
• All console errors from previous versions resolved
Next (v1.6)
Energy Net Filter Improvements
• Enhanced filter system for Energy Net (more granular options: specific modules, energy level ranges, creature age/size, etc.)
• Visual feedback when scooping (net physics, bunching behavior)
• Ability to save/load filter presets
Planned (v1.7+)
Treasure Room & Inventory System
• BS Treasure Store with 10+ high-cost nonsense items (gold statues, cursed relics, etc.)
• 10-slot player inventory that must be cleared before new purchases
• Dedicated Treasure Room where items can be tossed or carefully arranged
• Basic physics for piling treasure in the corner
• Future potential: use treasure as resources/ammo
